# Анализ лояльности пользователей Яндекс Афиши

## Описание проекта
Исследовательский анализ данных для определения факторов, влияющих на возврат пользователей на платформу Яндекс Афиша.

## Цель проекта
Выявить характеристики пользователей, которые с большей вероятностью возвращаются и делают повторные заказы.

## Задачи
- Анализ поведения пользователей
- Идентификация факторов, влияющих на лояльность
- Сегментация пользователей по активности  
- Разработка рекомендаций для маркетинговой команды

## Используемые технологии
- **Python** - основной язык программирования
- **Pandas** - обработка и анализ данных
- **SQLAlchemy** - подключение к базе данных
- **Matplotlib/Seaborn** - визуализация данных
- **Phik** - корреляционный анализ
- **Jupyter Notebook** - среда разработки
